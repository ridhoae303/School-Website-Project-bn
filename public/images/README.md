## How to Add Images

1. Create the appropriate subdirectory if it doesn't exist
2. Add your image files (JPG, PNG recommended)
3. Update the corresponding page component to reference the new image path
4. Example: `<img src="/images/home/hero-slide-1.jpg" alt="description" />`
